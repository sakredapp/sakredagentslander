2) Exact Brand Tokens → Web Implementation Spec

You gave exact values. Here’s how to implement them cleanly.

A) CSS variables (global)

Use this in globals.css (or similar):

:root {
  /* Fonts */
  --font-body: "DM Sans", system-ui, -apple-system, Segoe UI, Roboto, sans-serif;
  --font-headline: "Playfair Display", Georgia, serif;

  /* Light mode surfaces */
  --bg-root: #F9F9F7;
  --bg-card: #FFFFFF;
  --bg-input: #FFFEFA;

  /* Text */
  --text-primary: #0F172A;

  /* Gold system */
  --gold: #C5A059;
  --gold-light: #EBD598;
  --gold-dark: #A68A4A;
  --gold-subtle: rgba(197, 160, 89, 0.1);
  --gold-medium: rgba(197, 160, 89, 0.2);
  --gold-border: rgba(197, 160, 89, 0.3);

  /* Status */
  --success: #4A7C59;
  --error: #9B3B3B;
  --warning: #C5A059;

  /* Gradient */
  --gold-grad-start: #C5A059;
  --gold-grad-mid: #D4B76E;
  --gold-grad-end: #EBD598;
}

B) Tailwind mapping (recommended)

In tailwind.config.js, map to your system so dev doesn’t freestyle colors:

theme: {
  extend: {
    fontFamily: {
      sans: ['var(--font-body)'],
      serif: ['var(--font-headline)'],
    },
    colors: {
      bg: {
        root: 'var(--bg-root)',
        card: 'var(--bg-card)',
        input: 'var(--bg-input)',
      },
      text: {
        primary: 'var(--text-primary)',
      },
      gold: {
        DEFAULT: 'var(--gold)',
        light: 'var(--gold-light)',
        dark: 'var(--gold-dark)',
        subtle: 'var(--gold-subtle)',
        medium: 'var(--gold-medium)',
        border: 'var(--gold-border)',
      },
      success: 'var(--success)',
      error: 'var(--error)',
      warning: 'var(--warning)',
    },
    backgroundImage: {
      'gold-gradient': 'linear-gradient(90deg, var(--gold-grad-start), var(--gold-grad-mid), var(--gold-grad-end))',
    },
  },
}
